# QossBuddy – Quality of Service Management Platform

## Overview
This project was developed as part of the **PIDEV – 4rd Year Engineering Program** at **Esprit School of Engineering** (Academic Year 2025–2026).

QossBuddy is a full-stack web application designed to help organizations monitor, evaluate, and improve the **Quality of Service (QoS)** provided to their users.
